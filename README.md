# AI Powered Smart Cable Quality Inspection System

## Overview

This project uses MobileNetV2 Transfer Learning to classify cable images into:

- Good Cable
- Damaged Cable

The application provides:

- Image Upload
- AI Prediction
- Confidence Score
- Dashboard
- PDF Report
